# CIS2023_Final_Project

## Overview
A bash script designed to count from 1 to 50 and output to the terminal. This project was completed for CIS2023 at Pikes Peak State College.

## Documentation
For full project details, including screenshots of the build process and execution, please refer to:
* `Documentation/Final_Project_Documentation.odt`

## How to Run
1. Ensure the script has execute permissions: `chmod +x counter.sh`
2. Run the script: `./counter.sh`
